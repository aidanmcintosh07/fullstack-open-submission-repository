- Client tells the server to make a GET request to https://studies.cs.helsinki.fiexampleapp/notes
- The server returns a HTML document to the client

- Client tells the server to make a GET request to https://studies.cs.helsinki.fi/exampleapp/main.css
- The server returns a CSS file that styles page

- Client tells the server to make a GET request to https://studies.cs.helsinki.fi/exampleapp/main.js
- The server returns a JavaScript file to the client

- Client tells the server to make a GET request to https://studies.cs.helsinki.fi/exampleapp/data.json
- The server returns some data in json format which is logged to the browser console and displayed on the notes page
